# AI First Credit Union - Image Specifications

## Brand Guidelines
- **Primary Color**: Blue (#2563EB - equivalent to bg-blue-600)
- **Secondary Colors**: Gray (#374151 for dark, #6B7280 for medium, #9CA3AF for light)
- **Background**: White (#FFFFFF) or transparent
- **Style**: Modern, clean, professional with tech-forward aesthetic
- **Format**: PNG with transparency preferred, JPG for photos

## üé® **Design Guidelines**

### Color Specifications
- **Primary Blue**: #2563EB (RGB: 37, 99, 235)
- **Blue Hover**: #1D4ED8 (RGB: 29, 78, 216)
- **Dark Gray**: #374151 (RGB: 55, 65, 81)
- **Medium Gray**: #6B7280 (RGB: 107, 114, 128)
- **Light Gray**: #9CA3AF (RGB: 156, 163, 175)

### Style Requirements
- **Consistent Style**: All icons should follow the same design language
- **Professional Look**: Clean, modern, and trustworthy appearance
- **Tech Elements**: Subtle incorporation of AI/technology themes where appropriate
- **Accessibility**: High contrast ratios for visibility
- **Scalability**: Icons should look good at different sizes

### File Format Requirements
- **Icons**: PNG with transparent background
- **Photos**: JPG or PNG, optimized for web
- **Logo**: PNG with transparency, also provide SVG version if possible
- **Compression**: Optimize file sizes for web performance
